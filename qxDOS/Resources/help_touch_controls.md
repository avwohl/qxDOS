# Touch Controls

Virtual touch controls let you play DOS games on iPad without a physical keyboard. An overlay of buttons, D-pads, and analog sticks appears on top of the emulator screen.

## Getting Started

1. Tap the **gamecontroller** icon in the left sidebar to choose a layout
2. Select a built-in preset or create your own
3. Tap the **eye** icon in the sidebar to show or hide the overlay

Touch controls are hidden automatically on Mac where a keyboard is available.

## Built-in Presets

Four presets are included:

| Preset | Best for | Controls |
|--------|----------|----------|
| **DOOM** | DOOM, Heretic, Hexen | D-Pad + look stick + fire/use/strafe buttons + weapon keys 1-5 |
| **Duke Nukem 3D** | Duke3D, Shadow Warrior | D-Pad + look stick + crouch/jump/open/fire buttons + weapon keys |
| **General FPS** | Any first-person game | Same as DOOM with generic button labels |
| **Arrow Keys Only** | Adventures, puzzles, menus | D-Pad + Enter, Space, Y, N, Esc |

To use a preset, tap **Use** next to it. To customize a preset, tap **Edit** — this creates a copy you can modify.

## Control Types

### D-Pad

An 8-directional pad for movement. Drag your thumb in any direction, including diagonals. Each direction sends a configurable key press (arrow keys by default).

### Look Stick

An analog stick that sends relative mouse movement. Drag to look around or aim. A quick tap on the stick can trigger an action (fire by default). Good for mouselook in FPS games.

### Button

A single action button. Press and hold to send a key, release to stop. Displays the key label on screen. Use for fire, jump, use, or any single key.

### Button Row

A horizontal row of small buttons (2-5). Each button sends its own key. Use for weapon selection (1-5), function keys, or any group of related keys.

## Creating a Custom Layout

1. Open the touch control editor (gamecontroller icon)
2. Tap **New Layout** under "My Layouts"
3. Name your layout
4. Tap **Add Control** to add D-pads, sticks, or buttons
5. Configure each control's position, size, opacity, and key bindings
6. Use the preview at the bottom to see the layout
7. Tap **Save**

You can also duplicate a built-in preset as a starting point.

## Editing Controls

Each control has these settings:

- **Position** (X, Y) — where on screen, as a percentage (0-100%)
- **Size** — how large the control appears
- **Opacity** — how transparent the overlay is (10-80%)
- **Key Bindings** — which keys each direction or button sends

For D-pads, you set the key for each of the four directions. For button rows, you set the key for each button in the row. Any key on a standard keyboard can be bound, including function keys, modifiers (Ctrl, Alt, Shift), and special keys.

## Tips

- Lower opacity (20-30%) lets you see the game beneath the controls
- Place the D-pad on the left and the stick on the right for a console-like feel
- Use the **None** option to disable touch controls entirely
- Long-press a custom layout for rename, duplicate, and delete options
- The stick sends mouse movement, not keys — it works with any game that uses mouselook
- Button rows are compact: use them for weapon keys or other secondary controls along the screen edge
